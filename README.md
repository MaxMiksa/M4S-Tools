# Run and deploy thisapp


View this app in Gemini AI Studio: https://ai.studio/apps/drive/1bBh4Hi4XWLAzjC47PDUdHofpu6Aj8TVs

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to the Gemini API key
3. Run the app:
   `npm run dev`
